# Tea-Party-Web-and-Text-Analysis
Thoughts on the possibility of creating a dataset of Tea Party Movement websites and comments for quantitative analysis. This README has been written as part of a project for the free "Building AI" online course by Elements of AI. 

## Summary
My project is the quantitative analysis of comments and forum threads on websites associated with the Tea Party Movement in the United States around 2009-12.
   
## Background
For historians, the digital age has brought new problems and opportunities. With more and more people having access to the Internet and being able to share their views online the approach of “history from below” seems all the more interesting. But how can we use the vast amount of sources provided to us by websites, message boards, and social media?
   
## Data and AI techniques
The project depends on data derived from the comments, threads, and posts on Tea Party associated websites and message boards. There are hundreds of websites that are part of this network. There are two tasks. First, get the data from the websites by scraping them. Because the websites on the early Internet were very much eclectic, a scraper will have to be built for each website. To do this implementing a reinforcement learning algorithm that can learn to scrape the websites by itself is key. Secondly, the data has to be analysed. For this, Natural Language Processing and Word-Proximity Analysis could be the main approaches.
   
## How is it used
This is a very specialized use-case that will produce first, a very specialized data-set and then a very specialized insight into a certain movement at a certain time. But having build an automatic web-scraper, there is certainly the possibility to gather new data on different topics along the same website structure (threads, comments etc.).
   
## Challenges
The limitations of quantitative approaches to history are still discussed in the literature for now. There is a consensus that quantitative approaches alone are not able to tell us a lot about a topic. But in combination with close reading  of some of the source material patterns can be established and statements about the ideology, tactics, and organizational patterns of the Tea Party Movement could be established.

## What next
Looking ahead towards institutionalized message boards like 4chan and Reddit, the approach and skills developed in this project can be forwarded to develop further the quantitative analysis of the online Right.
